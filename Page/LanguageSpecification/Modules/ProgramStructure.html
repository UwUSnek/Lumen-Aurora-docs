All source code that makes up a program is contained within the root scope. The root scope can be thought of as an unnamed namespace. The current module is equivalent to an unnamed namespace inside the root scope.

Imported modules also reside in the root scope. They are equivalent to named namespaces containing the imported symbols. The __internal module is always imported automatically, as it contains the primitive types and fundamental functions.

File contents of the module
import io; void main { io.print(">:3"); }
Full program structure
// Root scope namespace { namespace __internal { // Primitive types and functions... } namespace io { void print(){ ... } } // Current module namespace { void main { io.print(">:3"); } } }
