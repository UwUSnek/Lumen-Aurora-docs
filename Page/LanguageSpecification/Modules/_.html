A module is a file containing Lumen or Aurora source code. Modules are used to organize the program into self-contained units.

By default, symbols declared in a module are not accessible from the outside. Modules can define which symbols are available to other modules through the export directive. These symbols can then be accessed using the import directive.

Each module can define a main function. This is where execution begins when the program is started, but only if the module was passed to the compiler as the entry point. This is covered in detail in Compilation.
