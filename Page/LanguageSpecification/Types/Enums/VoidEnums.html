Enums can use void as base type.
Elements of void enums don't have a base value and cannot be initialized explicitly.

Example
import io; enum Season : void { Spring; Summer; Autumn; Winter; } void main() { io.println("The " + Season:len + " seasons are:"); for(s in Season:values) { io.println("#" + (s.index + 1) + " " + s.name); } // io.println(Season.Autumn.value); Error, void enum has no value member }
Output
The 4 seasons are: #1 Spring #2 Summer #3 Autumn #4 Winter
