Enums are types whose allowed values are a named subset of the possible values of their base type. The allowed values are called Enum elements.
Unlike most languages, the base type can be any type, including pointers, structs, other enums or even void.

Enum elements work similarly to struct values, containing their name, index, and base value as members. They are part of the global initialization order and their base value is initialized using one of the base type's constructors.
If the base type defines a destructor, it is called on each element before the process ends.

Expressions of enum type are always 8 bytes in size. They are implicitly convertible to any type their base type is implicitly convertible to.
This is explained in detail in Implicit conversions.

Example
import io; enum TempThreshold : float { AbsoluteZero = -273.15; Freezing = 0; Boiling = 100; } void main() { io.println("Water boils at " + TempThreshold.Boiling.value + "°C"); }
Output
Water boils at 100°C

Enum elements are equivalent to global const variables, but enum values simply reference the existing elements. This means that creating or destroying enum values doesn't call any constructor or destructor of the base type.
