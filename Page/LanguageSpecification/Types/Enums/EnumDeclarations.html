Enums are declared using the enum keyword followed by its name and base type. Enum elements are declared in the same way as variables, but the type and the const qualifier are implicit and their initializer values must be constant expressions.

Elements of integral enums that are not initialized are given the value of the preceding element + 1, or 0 if they are the first element.
Elements of enums with any other base type must always be initialized.
Multiple elements can have identical values, but each identifier can only be declared once.
Empty enums are allowed.

Declaration syntax Element declaration syntax Example
import io; enum ErrorCode : uint { Success = 0; // General errors Unknown = 100; InvalidInput; // 101 OutOfMemory; // 102 // IO errors FileNotFound = 200; AccessDenied; // 201 DiskFull; // 202 } void main() { ErrorCode err = ErrorCode.FileNotFound; io.print("Error code: " + err); }
Output
Error code: 200
