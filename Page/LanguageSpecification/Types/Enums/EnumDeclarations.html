An enum is declared using the enum keyword followed by its name and base type. Enum elements are declared in the same way as variables, but the type is implicit and their initializer values must be constant expressions of the base type.

Elements of integral enums with no initializer value are given the value of the preceding element + 1, or 0 if they are the first element.
Elements of void enums cannot be initialized explicitly.
Elements of enums of any other base type must always be initialized.

Multiple elements can have identical values, but each identifier can only be declared once.
Empty enums are not allowed.

Declaration syntax Element declaration syntax Example 1
import io; enum ErrorCode : uint { Success = 0; // General errors Unknown = 100; InvalidInput; // 101 OutOfMemory; // 102 // IO errors FileNotFound = 200; AccessDenied; // 201 DiskFull; // 202 } void main() { ErrorCode err = ErrorCode.FileNotFound; io.print("Error code: " + err.value + " - " + err.name); }
Output
Error code: 200 - FileNotFound
Example 2
import io; struct ResolutionData { uint width, height; uint pixels; str ratio; init(uint w, uint h, str r) { width = w; height = h; pixels = w * h; ratio = r; } } str toString(ResolutionData r) { return "" + r.width + "x" + r.height + ", " + r.pixels + " pixels, " + r.ratio; } enum Resolution : ResolutionData { HD = { 1280, 720, "16:9" } FullHD = { 1920, 1080, "16:9" } QuadHD = { 2560, 1440, "16:9" } UltraHD = { 3840, 2160, "16:9" } } void main() { io.println("FullHD: " + Resolution.FullHD->toString()); }
Output
FullHD: 1920x1080, 2073600 pixels, 16:9
