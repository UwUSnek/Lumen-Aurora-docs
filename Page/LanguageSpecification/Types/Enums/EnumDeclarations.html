Enums are declared using the enum keyword followed by its name and base type. Unlike most languages, the base type can be any type, including structs or other enums.

Enum elements are declared in the same way as variables, but the type and the const qualifier are implicit and their initializer values must be constant expressions.

Elements of integral enums that are not initialized are given the value of the preceding element + 1, or 0 if they are the first element.
Elements of enums with any other base type must always be initialized.
Multiple elements can have identical values, but each identifier can only be declared once.
Empty enums are allowed.

enum e : int { a, b; // a = 0, b = 1 c = -266250; // c = -266250 d, e = 0; // d = -266249, e = 0 } float main(){ return e.c; // Implicitly convert e to float }
