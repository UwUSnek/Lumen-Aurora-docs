The drop function is a special struct method that is called each time a value of its parent struct type is destroyed.
It has no name and no return value and it's declared using the drop meta keyword.

Each struct can only have one drop function. It cannot be specialized and cannot take any argument, nor can it be called explicitly.
Templates and specialization are not allowed.
There is no default drop function.

Example
import <io>; int last = 0; struct s { float n; int i; init(float n){ i = last++; this.n = n; } init(){ io.print("Destroyed instance n." + str(i) + "\n"); } } void main(){ s var = { 0.5 }; s var2 = { 123 }; }
Output
Destroyed instance n.0 Destroyed instance n.1
