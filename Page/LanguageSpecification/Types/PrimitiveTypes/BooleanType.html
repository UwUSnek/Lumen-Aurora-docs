The Boolean type is used to represent truth values. They are ideal for storing conditions and flags.
true and false are the only values that can be stored in a bool.

Booleans are 4 bytes in size. They purposely waste 31 bits for performance reasons.
Smaller types would require additional bitwise operations and expensive conversions between CPU and GPU.

Unlike other primitive types, bool only has one variant.

Example
import io; void main() { while(true) { io.print("Insert a number: ") float n = io.read<float>(); bool p = n >= 0; io.println("" + n + " is a " + ("positive" if p else "negative") + " number.\n"); } }
