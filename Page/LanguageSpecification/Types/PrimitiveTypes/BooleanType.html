The Boolean type is used to represent boolean values.
true and false are the only values that can be stored in a bool.

Booleans are 4 bytes in size. They purposely waste 31 bits for performance reasons.
Smaller types would require additional bitwise operations and expensive conversions between CPU and GPU.
