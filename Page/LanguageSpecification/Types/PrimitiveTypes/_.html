Primitive types are the simplest kind of type. They are used to represent integral numbers, decimal numbers and boolean values. They are the foundation all other types and complex data structures are based on.

All primitive types are defined in the __internal module.
Their names are not keywords and can be shadowed like any other symbol.

The primitive types are:

Type name Size Value type
ubyte uint ulong ucent 1, 4, 8, and 16 bytes Unsigned integral number
byte int long cent 1, 4, 8, and 16 bytes Signed integral number
float double quad 4, 8, and 16 bytes Floating point number
bool 4 bytes Boolean value

All primitives types are convertible to each other. This is explained in detail in Implicit conversions.
