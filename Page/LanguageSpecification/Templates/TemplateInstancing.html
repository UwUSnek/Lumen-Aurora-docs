Templates are instanciated when needed.

Specific instances can be referenced by explicitly specifying the template arguments between agle brackets.
This disables argument type deduction.

Template instance path * Example 1
struct s { int n = 5; } template<type t> t add(t a, t b){ return a + b; } void main() { add<float>(5, 6); // 5 and 6 are converted to float add<s>({}); // add<s> cannot be instantiated without generating semantic errors float(float) fn = add<float>; fn(0.0, 6.7); }
