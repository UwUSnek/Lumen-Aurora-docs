Basic parameters are named memory locations used to pass arguments to routine calls.
They work in the same way as variables, but the initializer value is defined by the caller routine.
Just like variables, basic parameters are first allocated and then initialized by calling the = operator, one parameter at a time, following the order in which they were declared.

Declaration syntax Example
int add(uint a, uint b) { return a + b; } void main(){ uint a = 5; uint r = add(a, 6); }
