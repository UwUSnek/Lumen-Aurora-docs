Optional parameters are parameters that can specify a default value when no argument is provided by the caller.
Optional parameters are only available for functions and template functions.
A function can have multiple optional parameters, but they must all be declared after the last mandatory parameter.

Reference parameters cannot be optional.
Parameter packs also cannot be optional.

Declaration syntax Example
import <math>; double pow(double a, double b = 2) { return math.pow(a, b); } void main(){ pow(3, 3); // Returns 27 pow(3); // Returns 9 }
