Struct specialization allows structs to define different bodies based on constant conditions. This is done by specifying a specialization constraint, which contains the condition, and a corresponding body.

Struct declarations whose specialization constraint is not satisfied are discarded. As with routine specialization, if zero or more than one declaration is suitable, a semantic error is generated.

Example
template<type t> struct s when(t:id != void:id) { t n; constructor() { io.print("Not void\n"); } } template<type t> struct s when(t:id == void:id) { constructor() { io.print("Void\n"); } } void main(){ t<int> a; t<void> b; }
Output
Not void Void
