"Values" refers to any information stored in a typed container during runtime.
These include Variables, Routine parameters, Temporary variables, data returned by Routines, and the result of member access and pointer dereferencing.

Each value has a defined lifetime.
This usually goes from where they are declared to the end of their enclosing scope, though it can be extended using custom init and copy functions.
