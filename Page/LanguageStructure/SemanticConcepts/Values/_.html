"Value" refers to any information stored in a typed container during runtime.
These include Variables, Routine parameters, Enum elements, Temporary variables, and the result of member access and pointer dereferencing on non-temporary values.

Each value has a defined lifetime that starts when it's created and ends when execution reaches the end of its enclosing scope.
