Aurora is a specialized subset of Lumen focused on managing GPU tasks and shaders.
When developing applications for graphics or parallel computing, many languages require developers to manage complex GPU tasks manually. Aurora streamlines this process, providing a simplified interface while still allowing developers the flexibility to fine-tune any aspect of the GPU operations when necessary.

Aurora seamlessly integrates with Lumen.
This enables developers to work in a unified environment without the need for external tools.
