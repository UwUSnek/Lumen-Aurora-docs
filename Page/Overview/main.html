Lumen and Aurora are high-level programming languages designed to work together seamlessly, offering an integrated solution for both general-purpose programming and GPU computing.

Both languages are compiled and share a static type system, offering a balance between flexibility and performance without compromising code clarity or maintainability.

Lumen - Programming language. File extension: .lmn

Aurora - Shading language. File extension: .aur
