//TODO Lumen and Aurora are high level procedural compiled languages with the aim of abstracting GPU rendering while preserving the runtime performance of low level languages. This is achieved through a built-in rendering support and a powerful optimizer.

The languages are organized in modules and share the same compiler.
A static type system is used to improve readability and minimize runtime errors.
A reference counter manages the lifespan of variables and memory blocks.

Lumen - Programming language. File extension: .lum

Aurora - Shading language. File extension: .aur
